Environment needed

1) Anaconda (for all SciKit related packages)
2) Enchant (for eliminating stopwords)
3) NLTK - for stopwords (pip install nltk)
4) Download NLTK dictionary nltk.download()


How to Run

python classifier_main.py [a|b|c|d|e|f]

a - intelligrep
b - naive bayes
c - naive bayes improved1
d - naive bayes improved2
e - perceptron
f - tfidf


Developed with Python 3.6 on Windows
